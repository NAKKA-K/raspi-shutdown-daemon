# raspi-shutdown-daemon
SW7を5秒間押し続けると、シャットダウンされるプログラム

# Description
The LED1 glows, while pressing SW7.
Press and hold for 5 seconds to *****shutdown**.
When shutting down, the message will be alerted.

# Install
```
$ make install
```

# Requirement
- python3-pygame
- RPi.GPIO

# Usage
```
$ make install
$ make run
```
